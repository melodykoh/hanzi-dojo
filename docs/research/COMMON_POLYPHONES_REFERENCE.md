# Common Chinese Polyphones (多音字) - Quick Reference

**Purpose:** Reference guide for the most common multi-pronunciation characters in HSK 1-4
**Date:** December 9, 2025

---

## How to Use This Guide

Each entry shows:
- **Character** (Simplified/Traditional)
- **Pronunciations** (Pinyin + Zhuyin)
- **Meanings** per pronunciation
- **Context Words** with meanings
- **HSK Level** where it appears

## HSK 1 Polyphones

### 1. 长 / 長

**zhǎng** (ㄓㄤˇ) - to grow, elder, chief
- 生长 (shēngzhǎng) - to grow
- 校长 (xiàozhǎng) - principal
- 长大 (zhǎngdà) - grow up
- 长辈 (zhǎngbèi) - elder

**cháng** (ㄔㄤˊ) - long, length
- 长度 (chángdù) - length
- 长江 (Chángjiāng) - Yangtze River
- 很长 (hěn cháng) - very long
- 长城 (Chángchéng) - Great Wall

---

### 2. 好

**hǎo** (ㄏㄠˇ) - good, easy, well
- 很好 (hěn hǎo) - very good
- 好人 (hǎo rén) - good person
- 好吃 (hǎo chī) - delicious
- 好看 (hǎo kàn) - good-looking

**hào** (ㄏㄠˋ) - to like, to be fond of
- 爱好 (àihào) - hobby
- 好奇 (hàoqí) - curious
- 好学 (hàoxué) - eager to learn
- 爱好者 (àihàozhě) - enthusiast

---

### 3. 中

**zhōng** (ㄓㄨㄥ) - middle, center, China
- 中国 (Zhōngguó) - China
- 中间 (zhōngjiān) - middle
- 中午 (zhōngwǔ) - noon
- 中心 (zhōngxīn) - center

**zhòng** (ㄓㄨㄥˋ) - to hit (target), to succeed
- 中奖 (zhòngjiǎng) - win a prize
- 打中 (dǎzhòng) - hit the target
- 中毒 (zhòngdú) - be poisoned

---

### 4. 得

**de** (ㄉㄜ˙) - (particle after verb)
- 说得好 (shuō de hǎo) - speak well
- 做得到 (zuò de dào) - can do it
- 走得快 (zǒu de kuài) - walk fast

**dé** (ㄉㄜˊ) - to obtain, to get
- 得到 (dédào) - to obtain
- 获得 (huòdé) - to gain
- 得意 (déyì) - proud

**děi** (ㄉㄟˇ) - must, have to
- 得去 (děi qù) - must go
- 得做 (děi zuò) - must do
- 我得走了 (wǒ děi zǒu le) - I have to go

## Usage Tips

### For Learners

1. **Focus on context words** - The compound words show you WHEN to use each pronunciation
2. **Practice with HSK level** - Start with HSK 1-2 polyphones first
3. **Listen to audio** - Pronunciation is key for polyphones
4. **Use in sentences** - Don't just memorize lists, use them in context

### For Hanzi Dojo Integration

1. **Drill A guardrails** - Exclude all valid variants from distractors
2. **Context word hints** - Show compound word when testing alternate pronunciations
3. **Progressive disclosure** - Teach default pronunciation first, variants later
4. **Separate practice** - Consider "Variant Drill" mode for advanced learners
